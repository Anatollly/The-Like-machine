Инструкция по применению LIKE MACHINE

Различия между бесплатной и платной версией – в количественной возможности проставления LIKE в день и в ограничении сохраняемой информации для продвижения. Бесплатная версия 100 LIKE в день и по 3 сохраняемых GEO, HASH TAGS, поста и аккаунта. При оформлении подписки - это количество увеличивается до бесконечности.

  • Для начала настройте, как Вам нравится своё расширение - выберите цвет, эффект прокрутки, масштаб, расположение счетчика. Это делается для того, чтобы даже такая работа была Вам в радость. Обязательно сохраните их.
  • Выберите категорию для LIKE – ваша лента, конкретный аккаунт, нужный GEO или HASH TAG.
  • Проставьте количество желаемых LIKE, например, для начала 10, чтобы увидеть, как это работает.
  • Нажимаем START – Ваша LIKE MACHINE запущена!
  • Кнопками Pause и Stop можно приостановить, либо совсем остановить LIKEMACHINE

Работа с сохранением данных

  • Если Вам необходимо запомнить нужный GEO, HASH TAG, пост или аккаунт, для дальнейшей работы с ними, просто нажмите кнопку ##. LIKE MACHINE автоматически сохранит метку в нужную категорию.
  • Теперь можете зайти и проверить Избранное, там сохранены все Ваши метки.
  • При нажатии по метке Вы быстро переместитесь на нужную страницу

Скачать фото или видео контент

  • Зайдите на нужный вам пост
  • Нажмите кнопку Download
  • Контент автоматически загрузится к вам на компьютер

Кнопка TOP – предназначена для перемещения на самый верх страницы.

Ручной режим LIKE MACHINE – сделан для удобного и быстрого проставления LIKE, если Вам всё же необходимо это делать вручную.

  • Управляется пробелом, вводом и стрелками вниз, вверх, влево, вправо
  • Один пробел - переход к посту
  • Двойное нажатие пробела (быстроту нажатия вы регулируете в настройках) – LIKE
  • Повторное двойное нажатие пробела – заберут Ваш LIKE
  • Ввод (Enter) воспроизводит/останавливает видео
  • Стрелками можно перемещаться к следующему или предыдущему посту и просматривать посты с несколькими фото
